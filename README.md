# All-in-One Organizer

An all-in-one productivity web application that helps users manage their daily activities efficiently. This project combines multiple essential tools into a single platform, including a Notepad, To-Do List, Expense Tracker, and Income Tracker.

## 🚀 Live Demo

🔗 https://all-in-one-organizerr.onrender.com/

## 📌 Features

### 📝 Notepad

* Create and save notes
* Simple and user-friendly interface
* Quick access to important information

### ✅ To-Do List

* Add and manage tasks
* Track daily activities
* Mark tasks as completed

### 💰 Expense Tracker

* Record expenses
* Monitor spending habits
* Keep track of financial activities

### 📈 Income Tracker

* Record income sources
* Manage earnings efficiently
* Maintain a complete financial overview

## 🛠️ Technologies Used

## 🎯 Purpose

The goal of this project is to provide a centralized platform where users can:

* Organize notes
* Manage daily tasks
* Track expenses
* Monitor income

All from a single application.
